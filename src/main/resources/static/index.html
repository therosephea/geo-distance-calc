Geo Distance Calculator

Straight-line distance between two UK postal codes. All /api/** endpoints require HTTP Basic authentication.

Explore the API in Swagger UI →

Endpoints

GET/api/v1/distance?from={postcode}&to={postcode}

Distance in kilometres between two postcodes. Case and spaces are ignored.

GET/api/v1/postcodes/{code}

Coordinates for a single postcode.

PUT/api/v1/postcodes/{code}

Create or update a postcode's coordinates. Body: {"latitude": 57.2, "longitude": -2.2}

Example

curl -u wcc:$APP_PASSWORD \
  "http://localhost:8080/api/v1/distance?from=AB10%201XG&to=BB1%204HY"
{
  "start": { "postcode": "AB101XG", "latitude": 57.14414, "longitude": -2.114871 },
  "end":   { "postcode": "BB14HY",  "latitude": 53.7676,  "longitude": -2.417055 },
  "distance": 375.936,
  "unit": "km"
}

Responses

CodeMeaning
200Success
400Missing parameter, or a coordinate outside the valid range (±90 / ±180)
401Missing or incorrect credentials
404Postcode not found in the dataset
422Coordinates are valid but fall outside the UK service area
Set APP_PASSWORD before starting the service; the username is wcc. The OpenAPI specification is available at /v3/api-docs. See the project README for setup, configuration, and how to load the full postcode dataset.
